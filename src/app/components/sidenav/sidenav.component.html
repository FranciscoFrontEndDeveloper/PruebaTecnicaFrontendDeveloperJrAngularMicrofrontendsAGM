La API de Rick and Morty proporciona información detallada sobre los personajes, ubicaciones y episodios de la famosa serie animada. En esta aplicación, se utiliza la API para mostrar una tabla con los personajes, incluyendo su nombre, especie, género y estado.


Instrucción: haz clic en cualquiera de los elementos de la tabla para expandirlo y ver información adicional sobre el personaje seleccionado.
